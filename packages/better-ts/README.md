# better-ts

A better standard library for TypeScript: `Option`, `Result`, `Validated`,
`NonEmptyArray`, `DeepReadonly`, `Immutable`, `Lazy`, `Brand` and `Iter`.
**Zero dependency.** ESM only.

This is a thin meta package that re-exports everything from
[`@better-ts/core`](https://www.npmjs.com/package/@better-ts/core).

```bash
pnpm add better-ts
# equivalent to:
pnpm add @better-ts/core
```

```ts
import { Option, Result } from 'better-ts'
```

For the full documentation see
[`@better-ts/core`](https://www.npmjs.com/package/@better-ts/core).
